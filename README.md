# Quizzyverse

## Description
### This is a web application created using React as frontend and python's Flask framework as backend  

## Technologies used
- React
- Flask
- SQLite3

### Setup
- Install node
  - On Ubuntu
  ```bash
    curl -sL https://deb.nodesource.com/setup_10.x | sudo -E bash -
    sudo apt-get install -y nodejs 
    ```
    - On MacOS
    ```bash 
    brew install node
    ```
